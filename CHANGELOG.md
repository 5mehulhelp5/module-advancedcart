### Added
- Free shipping progress bar with configurable threshold and messages
- Quantity +/- buttons for cart items
- Trust badges section with 6 configurable badges
- Continue shopping button with customizable label and URL
- Cart savings display showing discounts and special price savings
- Estimated delivery date with business-day calculation
- Order notes with auto-save, persisting from cart to checkout to admin
- Enhanced empty cart page with illustration and trust badges
- Hyva and Luma theme support with automatic detection
- Admin order grid column for order notes
- Full admin configuration panel
